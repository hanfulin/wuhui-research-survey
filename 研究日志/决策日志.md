## 决策记录

### 2026-05-18：采用 Obsidian + GitHub + Firebase 工具组合
- **决策**：用这套组合支撑课题全过程
- **理由**：
  - GitHub 给报告和问卷网页做版本管理，每次修改都可回溯
  - Firebase 收集问卷数据，免费额度对医院内部调查规模完全够用
  - Obsidian 记录研究进展，Markdown 格式 Claude 能直接读写
- **依据**：《课题研究者项目指南-吴慧版.md》

### 2026-05-18：问卷采用匿名设计，去标识化
- **决策**：问卷不收集姓名/工号/手机号，只收部门、职称级别、工作年限区间
- **理由**：
  - 涉及医院员工对内部制度的评价，匿名能提高真实回答率
  - 数据会经 AI 分析，去标识化避免个人信息泄露风险
  - 按"部门×职称×年限"分类统计，对内部控制研究已足够
