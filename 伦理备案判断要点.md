# 伦理备案判断要点

> 用途：供吴老师向**南通大学附属医院伦理委员会**口头/书面咨询时使用的一页纸判断依据。
> ⚠️ 本文是判断辅助，**最终是否需要审查/备案以本院伦理委员会答复为准**。

---

## 一、本课题事实清单（咨询时直接报这几条）

| 维度 | 本课题实际情况 |
|------|---------------|
| 研究性质 | 管理学/内部控制研究，**非生物医学、非临床、不涉患者、不涉医疗干预** |
| 研究对象 | 本院在职员工（科研人员／管理人员／财务人员），均为**成年人、自愿** |
| 研究方法 | 问卷调查（问卷星扫码匿名填答）＋ 半结构化访谈（**现场笔录，不录音**） |
| 数据敏感性 | **全程去标识化**：不收集姓名／工号／手机号，仅存部门、职称级别、年限区间 |
| 知情同意 | 问卷首屏强制勾选知情同意；访谈口头宣读同意书后方可开始 |
| 生物样本 | 无（不涉及任何人体样本、基因、影像、病历数据） |
| 风险等级 | 不高于**最小风险**（仅为对本单位经费管理制度的看法，无身心健康风险） |
| 潜在敏感点 | 受访者可能对本单位管理提负面评价 → 已用匿名+去标识化保护，避免可追溯到个人 |

---

## 二、政策判断框架

**主要依据**：《涉及人的生命科学和医学研究伦理审查办法》（国卫科教发〔2023〕4号，2023-02-18 四部门联合印发）

判断逻辑分三档：

### 情形 A —— 大概率**需要伦理审查**
若本院将"涉及人的问卷／访谈调查"统一纳入伦理审查范围（很多三甲医院按此从严执行），则本课题**需提交伦理审查申请**，但通常可走**简易程序/快速审查**（理由：最小风险 + 去标识化 + 仅成人员工 + 无干预）。

### 情形 B —— 可申请**免除伦理审查**
《2023办法》第三十二条规定：使用**匿名信息数据**开展研究、对人体危害不大的可申请豁免。本课题"问卷星匿名扫码 + 去标识化 + 无干预 + 最小风险"较符合豁免情形，但**豁免与否仍须由伦理委员会作出决定**，研究者不能自行认定。

### 情形 C —— 课题主管单位另有要求 ✅ 已排除
经核对省卫健委原始通知《苏卫财务〔2026〕13号——关于组织申报2026年度卫生健康经济管理研究课题的通知》：
- 申报条件、课题管理、结题评审三节**均未要求任何伦理审查/备案证明**
- 结题只需按**附件3《江苏省卫生健康经济管理研究课题报告（电子版）编写规则》**提交报告
- **结论：省课题主管方无伦理证明要求**，卡点仅剩院内伦理委自身口径（情形 A/B）

## 三、给吴老师的行动清单

1. **先问院科研处/伦理委秘书**一句话：
   > "我们有个省级管理类课题，对本院员工做匿名问卷+访谈（不录音、去标识化、无干预），是否需要走伦理审查？能否走豁免或简易程序？"
2. 按答复分三种准备：
   - **要审查** → 备齐：课题申请书、问卷题目（v2 定稿）、访谈提纲、知情同意与匿名承诺文案 ＋ 填伦理审查申请表
   - **可豁免** → 索取一份**书面豁免说明/备案回执**（应对省课题结题查验）
   - **无需** → 让对方在邮件/OA 留一句书面确认，存档备查
3. ~~同步确认省卫健委课题口径~~ ✅ 已核实通知原文，省方不要求伦理证明，**无需再向省方确认**
