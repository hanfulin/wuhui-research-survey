## 记录

### 问题：本机无 GitHub CLI（gh），无法一键创建远程仓库
- **日期**：2026-05-18
- **现象**：初始化项目时，`gh` 命令未找到
- **原因**：电脑未安装 GitHub CLI
- **解决**：✅ 已解决。用 `winget install --id GitHub.cli` 安装（v2.92.0），
  装在 `C:\Program Files\GitHub CLI\`。之后 `gh auth login` 走浏览器授权登录。
- **预防**：换电脑工作前先确认 git / gh 都装好；gh 登录是一次性的，
  换电脑要重新 `gh auth login`
